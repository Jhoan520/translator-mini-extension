# Trasnlator Mini Extension
A simple translator extension for websites, primarily for Chromium-based browsers.

## Features
* Fast and simple translations.
* One-click copy to clipboard for all translations.
* Automatic Dark and Light support based on system settings.

## Technology & Credits
 * **Engine**: Powered by the Chromium Internal Translation Interface (GTX).
 * **Architecture**: This extension leverages the same internal endpoints used by Chromium-based browsers to provide seamless, lightweight translations without requiring a personal API Key.
 * **Third-Party Tool**: This is an independent development. It is NOT a Google product and is not affiliated with, sponsored by, or endorsed by Google LLC.

## Privacy Note
  * **Data Processing**: Translation requests are processed through Google's internal service endpoints. Your selected text is sent to these services solely for the purpose of translation, following the standard protocol of Chromium-based browsers.

## Tech Stack
* **JavaScript (ES6+):** For the core logic and API communication.
* **HTML5 & CSS3:** For the user interface and styling.
* **Chrome Extension API (Manifest v3):** The latest standard for secure and performant browser extensions.
